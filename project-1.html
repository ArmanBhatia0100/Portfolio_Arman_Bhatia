Gym Workout Tracker

A web application designed to help fitness enthusiasts log, track, and analyze their workouts efficiently with a seamless user experience.

Live Link

Project Overview

The Gym Workout Tracker is a personal project I built to assist fitness enthusiasts in managing their workout routines. This web app allows users to log exercises, including sets, reps, and weights, while providing tools like progress charts, rest timers, and goal-setting features to keep them motivated. I designed it with a focus on usability and performance, leveraging Next.js and React for a fast, seamless frontend experience, paired with a Node.js and Express backend for secure data handling.

The app includes user authentication with JWT for secure login and session management, and I opted for MongoDB to store workout data efficiently. Tailwind CSS powers the responsive, modern UI, ensuring it works flawlessly across devices. This project reflects my passion for creating practical, user-centric solutions—something I’ve honed through my experience as a Full-Stack Developer.
