Project Overview

The Project Dashboard is a web application I developed to streamline product inventory management. It allows users to add new products with details, view a comprehensive list of items, and delete products with ease—all wrapped in a clean, responsive UI. Built with React and Tailwind CSS, the frontend offers smooth navigation via React Router, while Material-UI components add a polished, professional look.

On the backend, I used Node.js and Express to handle CRUD operations, with MongoDB storing product data securely. I implemented backend validation to ensure data integrity and optimized performance with React best practices like lazy loading. This project showcases my ability to create scalable, user-focused tools—skills I’ve refined as a Full-Stack Developer.
